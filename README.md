# Калькулятор пряжи

Веб-приложение на Next.js для расчета метража при сложении нескольких нитей пряжи.

## Описание

Когда вы вяжете из нескольких нитей пряжи, сложенных вместе, важно знать итоговый метраж на 100 грамм. Этот калькулятор поможет вам рассчитать точный результат.

### Пример использования

У вас есть:
- Пряжа 1: 100г - 700м (берем в 2 сложения = 2 нити)
- Пряжа 2: 25г - 120м (берем 1 нить)

Итого: 3 нити сложены вместе

Калькулятор рассчитает итоговый метраж на 100г для такой комбинации.

## Как это работает

При сложении нескольких нитей пряжи вместе, итоговый метраж рассчитывается по формуле:

```
1/итог = 1/метраж₁ + 1/метраж₂ + ... + 1/метражₙ
```

Где каждый метраж приведен к 100 граммам и учитывает количество нитей.

## Возможности

- Добавление любого количества видов пряжи
- Указание количества нитей для каждого вида пряжи
- Автоматический расчет метража на 100г для каждой пряжи
- Расчет итогового метража при сложении всех нитей
- Адаптивный дизайн для мобильных и десктопных устройств
- Поддержка темной темы

## Установка и запуск

### Требования

- Node.js 18.17 или выше
- npm или yarn

### Установка зависимостей

```bash
npm install
```

### Запуск в режиме разработки

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

### Сборка для продакшена

```bash
npm run build
npm start
```

## Деплой на GitHub Pages

Проект настроен для автоматического деплоя на GitHub Pages через GitHub Actions.

### Настройка

1. Перейдите в Settings вашего репозитория на GitHub
2. В разделе "Pages" выберите:
   - **Source**: GitHub Actions
3. Отправьте изменения в ветку `main` или `master`
4. GitHub Actions автоматически соберет и задеплоит приложение

После успешного деплоя приложение будет доступно по адресу:
```
https://<ваш-username>.github.io/yarn-calc/
```

### Ручной запуск деплоя

Вы также можете запустить деплой вручную через вкладку "Actions" в репозитории.
